# SceneFlowNet
Implementation of '3D SceneFlowNet: Self-Supervised 3D Scene Flow Estimation'

Install the dependencies as:
```
conda dependencies/environment.yml
```

## Training SceneFlowNet
```bash
./run_flow_training.sh
```
## Testing SceneFlowNet
```bash
./run_flow_testing.sh
```
